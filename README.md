## What makes this site different

**Motion-first design.**  
Every section entrance, hover state, and transition is orchestrated with Framer Motion. Animations are purposeful — they guide attention, not distract from it. The page feels alive without feeling noisy.

**Technical SEO.**  
Metadata, Open Graph, structured data, semantic HTML, and optimized Core Web Vitals. Built to rank — not just to look good. An agency website that doesn't perform in search is a liability.

**Performance by default.**  
Next.js App Router with server components where possible. Images optimized via `next/image`. Fonts via `next/font`. Deployed on Vercel's edge network — sub-second load times globally.

**Built to convert.**  
Pricing is visible, the CTA is persistent, and the contact flow is frictionless. Every section is designed around one goal: turn a visitor into a client inquiry. Portafolio → Servicios → Nosotros → Contacto — in that order for a reason.

## Engineering decisions worth noting

**App Router over Pages Router.**  
Chose Next.js App Router from the start — server components give real performance benefits for a content-heavy marketing site. Less JavaScript shipped to the client means faster paint times, which directly affects SEO scoring.

**Framer Motion with purpose.**  
Animation on an agency site is a double-edged sword — too little and it feels static, too much and it feels amateur. Every animation has a job: section entrances establish hierarchy, hover states confirm interactivity, and the persistent scroll-to-top reinforces navigation control.

**Pricing transparency as a differentiator.**  
Most agencies in the Dominican Republic don't publish prices. NorthCode shows pricing upfront — a deliberate positioning decision that filters out low-budget leads and builds trust with serious clients before any conversation happens.
